Самая сложная из прочитанных мной технических книг, на которую я потратил целый месяц. Видимо еще не готов полностью осмыслить и через какое-то время нужно будет перечитать. Книга покрывает Java 5.0 и местами 6.0. Fork/Join из семерки отсутствует, лямбды тем более.

Пока читал, заметил у себя синдром студента-медика: начал в коде замечать проблемы синхронизации, места, которые лучше переписать не изобретая свои велосипеды, что, несомненно, говорит о пользе книги. Очень много времени посвящено Java Memory Model, вопросам видимости переменных и последствиям неправильной синхронизации. Итог: книга важна, но неподготовленный читатель уснет на введении.
